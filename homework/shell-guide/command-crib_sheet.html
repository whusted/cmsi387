    • Example: cd homework/shell-guide Short for "Change directory", cd (optionally) takes one argument: the name of the desired directory the user wishes to navigate to. cd alone will take the user to his or her home directory.

    Example: ls Short for "List", ls lists all the files in the user's current directory. Optional arguments are available, such as "-l", which lists files in "long format" (size, date modified, etc.)

    Example: mkdir new_folder Short for "make directory", mkdir does just that. The required argument after mkdir is the name of the new directory. This command is the same as creating a new folder in a Graphical User Interface.

    Example: cat example.txt anotherexample.txt Short for "Catenate", cat takes at least one argument (in the form of a text file) and concatenates the file(s) and prints to standard output.

    Example: ssh whusted@my.cs.lmu.edu Short for "Secure Shell", ssh is a progam for logging into a remote machine and for executing commands on a remote machine. LMU students will most likely use ssh in the above example.

    Example: top Short for "Table of Processes", the top command displays to standard output an ordered list of running processes, updating periodically if left running. Pragmatically it can be used to see if a certain application is using a lot of the CPU.

    Example: kill 1111 "Kill" is a command that is used in several popular operating systems to send signals to running processes in order to request the termination of the process. The argument is the process's ID. Like many other commands, this one does what it sounds like it does.

    Example: grep willy testfile.txt "Grep" is a utility for searching plain-text data sets for lines matching a regular expression. In the example above, "willy" is the regular expression being searched for in testfile.txt. If found, the expression is highlighted and printed to standard output. If not found, nothing is printed to standard output.

    Example: nice -20 test.java "nice" is used to invoke a utility or shell script with a particular priority, thus giving the process more or less CPU time than other processes. -20 (used in the above example) is the highest priority a process can have.

    Example: sudo rm test.txt sudo is a program for Unix-like computer operating systems that allows users to run programs with the security privileges of another user (usually this is the superuser). sudo requires a password to be performed; it's like a Trump card. Here is a great example of how to use sudo.

    Example: rm test.txt Short for "remove", rm removes the file whose name is given as the argument. In the above example, test.txt is removed from the current directory (if it is in fact present).
